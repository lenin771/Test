# Инструкция для работы с Markdown

## Выделение текcта

Чтобы выделить текст курсивом необходимо его обрпмить звездочкой (*) пример: *текст*

Чтобы выделить текст полужирным необходимо его обрпмить двумя звездочками (**) пример: **текст**
